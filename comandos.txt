Problema de permissão no arquivo. Dentro do container `otto`, rode:

```bash
chmod 666 /ros2_ws/src/otto_description/scripts/walk.py
```

Ou pelo WSL, fora do container:

```bash
chmod 666 ~/otto/ros2_ws/src/otto_description/scripts/walk.py
```

Depois o VSCode consegue salvar normalmente.
